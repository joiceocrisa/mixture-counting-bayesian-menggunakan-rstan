**Mixture Counting** digunakan untuk menguji distribusi apa yang dimiliki oleh suatu data (y). 

Pada code ini, distribusi yang diuji adalah Distribusi Normal, Distribusi Gamma, dan Distribusi Eksponensial yang akan dibandingkan berdasarkan Likelihood yang diperoleh. 

**rstan** adalah package R untuk bahasa pemodelan probabilistik Stan, memungkinkan pengguna melakukan inferensi Bayesian di RStudio dengan antarmuka yang mudah digunakan untuk analisis data kompleks menggunakan MCMC dan optimasi, diinstal melalui install.packages("rstan") di R.
